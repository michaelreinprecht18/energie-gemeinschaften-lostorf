# Design-Entscheidungen

Richtung: **Naturstein** — warm, geerdet, nahbar, handwerklich.
Entschieden am 2026-07-05.

## Farben

| Rolle            | Hex       | Einsatz                                      |
|------------------|-----------|----------------------------------------------|
| Hintergrund      | `#E7F9E4` | Seiten-Hintergrund, helles Mintgrün          |
| Text (primär)    | `#1A1510` | Headlines, Fließtext                         |
| Text (sekundär)  | `#5C5248` | Untertitel, Labels, Fließtext hell           |
| Akzent (Gold)    | `#9A7B2E` | Icons, Labels, Highlights – sparsam einsetzen  |
| Button (primär)  | `#9DD295` | Haupt-CTA, gedämpftes Grün                     |
| Rahmen / Trennlinie | `#D9CEB5` | Divider, Card-Borders, dezente Linien     |
| Weiß (Cards)     | `#FFFFFF` | Card-Hintergründe, helle Sektionen          |

## Typografie

| Rolle       | Schrift            | Gewicht  | Anmerkung                         |
|-------------|-------------------|----------|------------------------------------|
| Headlines   | Libre Baskerville  | 700 bold | Kräftiger, erdverbundener Serif    |
| Subheadlines| Libre Baskerville  | 400      | Italic als Variante möglich        |
| Fließtext   | Nunito Sans        | 300/400  | Leicht, lesbar, freundlich         |
| Labels/Meta | Nunito Sans        | 400      | Grossbuchstaben mit Letter-Spacing |

Quelle: Google Fonts (beide kostenlos, DSGVO-konform wenn self-hosted)

## Layout / Stil-Prinzipien

- **Viel Weißraum** — großzügige Abstände, Luft zwischen Sektionen
- **Klare Sektionen** — abwechselnd `#F7F3EC` und `#FFFFFF` als Hintergrund
- **Gold sparsam** — nur für 1–2 Akzente pro Sektion (Button, Unterstrich, Icon)
- **Keine harten Schatten** — maximal dezente Box-Shadows oder gar keine
- **Mobile first** — Layout beginnt einspalitig, wächst auf Desktop

## Referenzen / Inspiration

- Stimmung: lokales Schweizer Handwerk, Bergwelt, Qualität ohne Prahlerei
- Kein Tech-Startup-Look, kein grelles Solar-Orange
