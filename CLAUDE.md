# Arbeitsanweisungen für Claude

Dieses Projekt ist eine einseitige Landing Page für Solar Lostorf.

## Kontext
- Einfaches HTML/CSS/JS ohne Framework (kein React, kein Build-Tool)
- Alle inhaltlichen Entscheidungen stehen in `projekt.md`
- Alle Design-Entscheidungen stehen in `design.md`
- Texte/Copy liegen in `content/`, getrennt vom Code

## Regeln
- Keine Frameworks oder Build-Tools einführen ohne Rückfrage
- CSS in `styles/`, JavaScript in `scripts/`
- Bilder und Icons in den jeweiligen `assets/`-Unterordnern ablegen
- Änderungen am Inhalt zuerst in `projekt.md` oder `content/` abstimmen
